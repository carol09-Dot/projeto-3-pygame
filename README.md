## 💻 Sobre o Projeto

**O Despertar da Cobra** transforma a jogabilidade tradicional do "Snake Game" em uma experiência mais tensa e temática. Em vez de uma cobra comum comendo frutas, você controla uma criatura sombria que se alimenta de almas perdidas. À medida que avança, a cobra deixa um rastro de sangue pelo cenário, aumentando o desafio e a imersão visual do jogo.

Este projeto foi desenvolvido com foco no aprendizado de lógica de programação, manipulação de eventos e desenvolvimento de jogos 2D.

## ⚙️ Funcionalidades e Mecânicas

- **Crescimento:** A cada alma consumida, a cobra aumenta de tamanho.
- **Sistema de Pontuação:** Cada alma coletada adiciona **10 pontos** ao seu contador.
- **Contador Temático:** Painel no jogo exibindo a quantidade de **"Almas Coletadas"**.
- **Efeito Visual:** Rastro persistente de sangue deixado pela cobra ao se movimentar.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Python 3
- **Biblioteca Principal:** [Pygame](https://www.pygame.org/)

## 🚀 Como Executar o Jogo

### 📌 Pré-requisitos

Para rodar o jogo, você precisa ter instalado em seu computador:
- [Python 3.x](https://www.python.org/)
- A biblioteca **Pygame** (instalada via terminal)

- ### 🎲 Passo a Passo

```bash
# 1. Clone este repositório
$ git clone [https://github.com/seu-usuario/projeto-3-pygame.git](https://github.com/seu-usuario/projeto-3-pygame.git)

# 2. Acesse a pasta do projeto
$ cd projeto-3-pygame

# 3. Instale a biblioteca Pygame
$ pip install pygame

# 4. Execute o jogo
$ python main.py
